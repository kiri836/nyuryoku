## Nyuryoku, a built from scratch, corne style keyboard.

Features:
- ~38% keyboard, a total of 40 keys and 4 rotary encoders
- Runs ZMK firmware
- Completely wireless, using 1250mah batteries meaning +3 weeks of on time without needing to charge
- Split, column staggered layout
- Has a battery switch for long term storage
- 3d printed casing allows for extra customization
